# 头像图片说明

## 如何替换头像

1. **准备头像图片**
   - 推荐尺寸：400x400 像素或更高
   - 推荐格式：JPG、PNG 或 SVG
   - 建议使用正方形图片以获得最佳显示效果

2. **替换方法**
   - 将你的头像图片文件命名为 `avatar.jpg`、`avatar.png` 或 `avatar.svg`
   - 放置在 `src/assets/images/` 目录下
   - 替换现有的头像文件

3. **更新配置**
   - 在 `src/data/texts.ts` 文件中修改头像路径：
   ```typescript
   hero: {
     avatar: {
       src: '/src/assets/images/your-avatar.jpg', // 修改为你的文件名
       alt: '你的名字头像',
       description: '你的个人描述'
     }
   }
   ```

## 当前头像

- **默认头像**: `avatar.svg` (简单的灰色占位符)
- **备用头像**: `avatar.jpg` (占位符文件)

## 头像特性

- **响应式设计**: 在移动端和桌面端都有合适的尺寸
- **圆形显示**: 自动裁剪为圆形
- **悬停效果**: 鼠标悬停时有缩放和边框效果
- **在线状态**: 右下角有绿色在线状态指示器
- **错误处理**: 如果图片加载失败，会显示默认的 SVG 头像
